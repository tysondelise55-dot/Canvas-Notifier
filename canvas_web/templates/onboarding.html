How to get your Canvas API token:

  1. Log in to Canvas and click your profile picture (top-right)
  2. Go to Settings
  3. Scroll down to Approved Integrations
  4. Click + New Access Token
  5. Name it "Canvas Assistant" and click Generate Token
  6. Copy the token and paste it above

Add your AI key

The assistant uses Claude AI. You'll need a free Anthropic API key.

How to get your Anthropic API key:

  1. Go to console.anthropic.com
  2. Sign up for a free account (or log in)
  3. Click API Keys in the left sidebar
  4. Click Create Key and give it a name
  5. Copy the key — you can only see it once!
  6. Paste it above
