How to get your Canvas API token:

  1. Log in to Canvas and click your profile picture (top-right)
  2. Go to Settings
  3. Scroll down to Approved Integrations
  4. Click + New Access Token
  5. Name it "Canvas Assistant" and click Generate Token
  6. Copy the token and paste it above

Add your AI key

Canvas Assistant uses OpenRouter so you can pick any AI model — Claude, GPT-4o, Gemini, and more.

How to get your OpenRouter API key:

  1. Go to openrouter.ai and sign up for a free account
  2. Click your profile icon (top-right) → API Keys
  3. Click Create Key, give it a name like "Canvas Assistant"
  4. Copy the key — you can only see it once!
  5. Paste it above

OpenRouter gives you access to Claude, GPT-4o, Gemini, and dozens of other models from one key. Many models have a free tier.
